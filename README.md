## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Raghaverma/react-tut.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open in your default browser at `http://localhost:3000`.
